Pale-O Ale-To Web Design

Role: Designer For: HCI 525 Date: Spring 2018

Background

Pale-O Ale-To was a concept for a website of a brewing company. This was created for my Hypermedia class to mimic what creating a wireframe as a user experience designer would do. This was supposed to be a low fidelity wireframe, only the home page was to be detailed and the rest was to be made up of placeholders. The wireframe was created in Axure and there is a home page, an about us page, a news/events page, a shop page, and a contact page. The images below show the different pages.
